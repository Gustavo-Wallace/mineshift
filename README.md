# Mineshift

Mineshift is a minesweeper roguelike about patterns, risk and rule manipulation.

Built with Godot 4.7 and GDScript. The current prototype implements the complete classic Minesweeper foundation: a safe first move, flags, empty-region expansion, victory, defeat, timer, mine counter, and repeatable resets.

## Controls

- Left click: reveal a cell
- Right click: place or remove a flag
- `R`: start a new field
- `NEW FIELD`: start a new field

## Run

Import `project.godot` in Godot 4.7 and press **F6** or **F5** to run the main scene.
