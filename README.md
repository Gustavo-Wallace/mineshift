# Mineshift

Mineshift is a minesweeper roguelike in development, built with Godot 4.7 and GDScript.

The current foundation is a five-field run with progressively larger boards, classic minesweeper rules, safe openings, flags, chords, compact transitions, and a board-first minimalist interface. Future systems will focus on manipulating board actions, information, risks, and rules.

## Controls

- Left click: reveal a cell or chord a revealed number
- Right click: place or remove a flag
- `R`: restart the current field
- `Esc`: pause

Import `project.godot` in Godot 4.7 and press **F6** or **F5** to run.
